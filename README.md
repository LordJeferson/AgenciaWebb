## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
AgencyWeb é um projeto desenvolvido durante o curso de Desenvolvimento Web Front-End com o objetivo de praticar os fundamentos da criação de páginas web. Ao longo do desenvolvimento, foram aplicados conceitos de HTML5, CSS3 e semântica web, explorando a estruturação correta do conteúdo, a organização dos elementos da página e a estilização da interface. O projeto serviu como base para compreender boas práticas de desenvolvimento front-end, acessibilidade e construção de layouts responsivos, contribuindo para o aprimoramento das habilidades essenciais na criação de sites modernos.
